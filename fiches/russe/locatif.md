# Russe: Locatif
> Permet d'indiquer le lieu
> S'emploie avec tout les verbes de position (sauf продолжать)

#### Construction
| Russe | Français |
| --: | :-- |
| в |  à, dans |
| на | sur  |
| о | à propos de |

|   | Singulier | Pluriel |
|---|---|---|
| -*voyelle* | remplacer par **e** | remplacer par **ах** |
| -*consomne* | ajouter **e** | ajouter **ax**

#### Exemples
Ком **на** дива**е**
Артист **в** театр**е**
